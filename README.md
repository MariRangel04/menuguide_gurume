# 🍣 MenuGuide Gurumê

Sistema web desenvolvido para auxiliar garçons e funcionários de restaurantes japoneses na consulta rápida de informações sobre pratos, ingredientes, alergênicos, molhos e toppings.
Projeto desenvolvido como parte da disciplina de **Soluções Computacionais** da **Universidade Católica de Brasília (UCB)**.


## 📖 Sobre o Projeto

Em restaurantes japoneses, os cardápios costumam possuir uma grande variedade de pratos, ingredientes e combinações, o que pode dificultar o treinamento de novos funcionários e gerar dúvidas durante o atendimento ao cliente.
O **MenuGuide Gurumê** foi criado para centralizar essas informações em uma plataforma digital intuitiva, permitindo consultas rápidas e organizadas durante o atendimento.


## 🎯 Objetivos

* Facilitar o treinamento de novos funcionários;
* Reduzir erros na descrição dos pratos;
* Melhorar a qualidade do atendimento;
* Exibir informações detalhadas sobre os pratos;
* Identificar ingredientes alergênicos;
* Organizar pratos por categorias;
* Permitir busca rápida de informações.

## ✨ Funcionalidades

### Cardápio Digital

* Visualização de pratos cadastrados;
* Organização por categorias;
* Interface intuitiva e responsiva.

### Busca de Pratos

* Pesquisa por nome;
* Localização rápida de informações.

### Detalhes dos Pratos

* Nome do prato;
* Descrição;
* Ingredientes;
* Molhos;
* Toppings;
* Informações sobre alergênicos.

### Gerenciamento

* Cadastro de pratos;
* Edição de pratos;
* Exclusão de pratos;
* Atualização de informações.

### Autenticação

* Login de usuários;
* Controle de acesso administrativo.
